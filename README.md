## Проект: Intop

**Описание:**  
Intop — это платформа для продажи товаров и услуг в Азербайджане, объединяющая продавцов и покупателей в одном удобном мобильном приложении. Приложение позволяет быстро находить нужные товары, оформлять заказы и связываться с продавцами напрямую.

**Особенности:**  
- Каталог товаров и услуг с удобной навигацией  
- Фильтры и поиск по категориям  
- Система заказов и онлайн-платежей  
- Прямое общение с продавцами через встроенный чат  
- Удобный UI/UX для быстрого и комфортного взаимодействия
